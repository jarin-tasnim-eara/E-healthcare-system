# E-healthcare-system
##Overview

A simple Java-based e-healthcare system that allows patients to schedule appointments with specialized doctors (eye or skin). The system manages patient details, doctor selection, appointment scheduling, and confirmation.

Features
Patient information input and management
Doctor selection based on check type (eye/skin)
Appointment scheduling and confirmation
Use of OOP concepts (abstraction, inheritance, encapsulation, polymorphism)
Concepts Used
Abstract class (MediPro)
Interface (Confirm)
Inheritance (Doctor, EyeDoctor, SkinDoctor)
Encapsulation (private attributes with getters)
Polymorphism (method overriding)
##Technologies

Java, NetBeans IDE

##Usage

Run the main class, enter patient details, and choose check type to schedule and confirm an appointment.

##Description

This project demonstrates the implementation of object-oriented programming concepts in a healthcare system using Java.
